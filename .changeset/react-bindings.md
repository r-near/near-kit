Initial release of @near-kit/react package with React bindings for near-kit:

- `NearProvider` and `useNear()` for context management
- `useAccount()` for account state
- `useBalance()` for balance fetching
- `useView()` for contract view calls
- `useAccountExists()` for account existence check
- `useCall()` for contract change calls
- `useSend()` for NEAR transfers
- `useContract()` for typed contract instances
- `useContractView()` for typed view calls
